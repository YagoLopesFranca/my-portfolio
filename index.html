Este projeto simula uma interface de celular com links para redes sociais. Desenvolvido com HTML e CSS, utilizei a tag <iframe> para carregar o conteúdo das mídias dinamicamente, sem recarregar a página. O objetivo foi praticar a criação de interfaces interativas e a navegação entre telas.
